If a memory should appear more often:

1. Add clear glossary terms.
2. Narrow the disclosure trigger.
3. Split mixed-topic memories into smaller memories.
4. Use a specific path name.
5. Check recall analysis after a real query.

If a memory appears too often:

1. Remove broad glossary terms.
2. Make the disclosure trigger more specific.
3. Lower the priority.
4. Split broad notes into narrower memories.
