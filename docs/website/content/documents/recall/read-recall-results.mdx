The score tells you how strongly Lore thinks a memory matches the task. Higher scores usually mean stronger relevance.

A recalled item can still be ignored when it clearly misses the task. A low-scoring item can still be useful when the path is exactly what you need.

Use the Recall page when you want to inspect:

| Question | Where to look |
|---|---|
| Why did this memory appear? | recall detail page |
| Which query found it? | query list |
| Which candidates were shown? | recall analysis |
| Which candidates were opened? | usage/adoption state |
