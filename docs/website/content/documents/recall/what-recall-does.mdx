Recall answers one question: “Which memories are relevant right now?”

It can match by:

| Signal | What it helps with |
|---|---|
| Path and exact terms | project names, tool names, URI pieces |
| Glossary terms | product aliases, abbreviations, repeated phrases |
| Meaning | related concepts even when wording changes |
| Recent use | active project context and recently changed memories |

Recall usually returns a small list. The agent should open useful candidates before relying on them.
