Write a memory when the information has future value.

| Situation | Write a memory? |
|---|---|
| A project rule will apply again | Yes |
| A user preference should shape future replies | Yes |
| A bug root cause explains future behavior | Yes |
| A one-off command output was only useful today | Usually skip |
| A draft idea may change in ten minutes | Usually wait |

Start small. Add one clear memory for one clear topic. Lore works best when each memory has a practical future trigger.
