Start with these settings:

| Setting area | What to configure |
|---|---|
| Server URL | the address used by browser and agent runtimes |
| API token | bearer token for protected access |
| Embedding model | endpoint used for semantic recall |
| View LLM | endpoint used for memory summaries, boot drafts, and Dream |
| Backup | local or remote backup target |

Save settings from the Web console when possible. Environment variables are useful for deployment defaults and automation.
