Recall settings affect what the agent sees.

| Setting | Practical effect |
|---|---|
| display threshold | how strong a memory must be before it appears |
| max display items | how many memories can be injected into context |
| exact/glossary/dense weights | which match signals matter most |
| query length limit | protects agent workflows from oversized prompts |
| timeout | returns a notice when recall takes too long |

Keep display count small at first. Three focused memories usually beat a long noisy list.
