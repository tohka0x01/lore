| Problem | What to check |
|---|---|
| Web console does not open | server address, container status, port, reverse proxy |
| Setup stays incomplete | embedding endpoint, View LLM endpoint, boot memory save status |
| Health check fails | database connection, server logs, environment values |
| Login or API calls fail | API token, bearer header, browser cookie |
