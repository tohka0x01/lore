| Problem | What to check |
|---|---|
| Agent receives no boot context | plugin install, runtime restart, server URL |
| Agent receives no recall block | prompt hook, API token, recall settings |
| MCP tools fail | MCP URL, client config, auth token |
| Agent opens stale memories | memory content, path moves, Dream changes, recent edits |

When reconnecting an agent, verify the Web console first, then restart the agent runtime.
