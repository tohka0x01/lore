| Problem | What to check |
|---|---|
| A memory is missing | glossary terms, disclosure trigger, path name, recall threshold |
| Too many memories appear | broad glossary terms, broad disclosure, high priority |
| Recall is slow | model endpoint latency, timeout setting, query size |
| Recall result is hard to understand | open Recall Drilldown and inspect the query detail |

Most recall problems improve by tightening memory shape: one topic, clear trigger, useful glossary terms.
