Supported runtimes include:

| Runtime | Typical use |
|---|---|
| Codex | coding agent sessions |
| Claude Code | terminal coding sessions |
| Pi | assistant runtime with extension support |
| OpenClaw | desktop agent workflow |
| Hermes | Python-based agent workflow |
| Generic MCP | any MCP client that can call Lore tools |

Use the runtimes you already work in. The value comes from sharing the same durable memory across them.
