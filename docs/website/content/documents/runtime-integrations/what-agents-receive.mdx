Connected agents can receive:

| Context | Purpose |
|---|---|
| Boot memory | stable rules, user preferences, and runtime guidance |
| Recall candidates | task-relevant memory pointers |
| MCP tools | browse, search, create, update, move, and delete memories |
| Usage tracking | whether a recalled memory was opened and used |

A healthy agent workflow is: recall candidates, open relevant memories, do the task, then write durable changes back.
