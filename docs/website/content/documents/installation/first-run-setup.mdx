After install, open the Web console and complete setup:

1. Configure the embedding model endpoint.
2. Configure the View LLM endpoint.
3. Save boot memories.
4. Confirm runtime-specific memories.
5. Open Settings and verify recall, Dream, backup, and auth settings.
6. Restart connected agent runtimes.

The embedding endpoint powers semantic recall. The View LLM improves generated memory summaries and Dream maintenance.
